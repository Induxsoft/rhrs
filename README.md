# Rhrs (Recursos humanos, reclutamiento y selección).
El paquete Rhrs (Recursos humanos, reclutamiento y selección) de V12 es el módulo que permite gestionar de forma integral los procesos de reclutamiento, selección y administración de recursos humanos, ofreciendo herramientas para campañas de captación de personal, seguimiento de postulantes y control de requisiciones. Su propósito es optimizar la incorporación de talento, garantizando trazabilidad y eficiencia en cada etapa del proceso.
